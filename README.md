# BrainMapRevision

**An open-source, customizable revision platform for students to review and master all subjects effectively.**

## Description

BrainMapRevision is a flexible and community-driven platform designed to help students prepare for exams across all subjects. With an easy-to-use interface, comprehensive study materials, interactive quizzes, and the ability to create your own revision boards, it empowers learners to tailor their study sessions to fit their needs.

This project is open source, welcoming contributions, and aims to provide detailed documentation to assist users and developers alike.

## Features

- Create and customize your own revision boards
- Access subject-specific study materials
- Interactive quizzes and flashcards
- **Exam board past paper questions with revision guides** - Practice with real questions from official exam boards, supported by comprehensive guides that explain the concepts and techniques needed
- Progress tracking and analytics
- User-friendly interface
- Fully open source with community contributions

## Past Paper Questions & Revision Guides

BrainMapRevision incorporates authentic past paper questions from various exam boards to provide students with realistic exam practice, accompanied by comprehensive revision guides to help you understand and master each topic. Our question bank includes:

- **Pre-loaded questions with revision guides**: A curated collection of past paper questions from major exam boards (AQA, Edexcel, OCR, WJEC, SQA, and others) already integrated into the platform, each accompanied by detailed revision guides that explain the concepts tested
- **Community contributions**: Users can submit additional past paper questions along with their own revision guides to expand the database and help fellow students
- **Topic-tagged questions**: All questions are organized by subject, topic, and difficulty level for targeted practice
- **Mark schemes and explanations**: Most questions come with official mark schemes, examiner guidance, and step-by-step revision guides explaining how to approach similar questions

### Contributing Past Paper Questions & Revision Guides

We encourage users to contribute past paper questions they've encountered along with revision guides to help other students. When adding questions, please:

1. Verify the question is from an official exam board past paper
2. Include the exam board, year, paper number, and question number
3. Add the mark scheme and any examiner reports if available
4. **Create or include a revision guide** that explains the topic, key concepts, and approach to answering similar questions
5. Tag the question with relevant topics and difficulty level
6. Ensure compliance with fair use and educational purposes

See our [Past Paper Contribution Guide](CONTRIBUTING.md#past-paper-questions) for detailed instructions.

## Getting Started

### Prerequisites

- Node.js and npm
- (Any other dependencies or setup instructions)

### Installation

1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/BrainMapRevision.git
   ```

2. Install dependencies:
   ```bash
   cd BrainMapRevision
   npm install
   ```

3. Run the development server:
   ```bash
   npm start
   ```
    
### Usage
 
- Browse available revision guides
- Create your own revision boards
- Add and organize study materials
- **Practice with past paper questions and study the accompanying revision guides** filtered by exam board and topic
- Share your boards with the community
